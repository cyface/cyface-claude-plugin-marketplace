# Privacy Policy

**ui-design-principles** Claude Code Plugin

Last updated: March 2, 2026

## Overview

The ui-design-principles plugin is a collection of static reference files (Markdown) that provide UI/UX design guidance within Claude Code. It does not collect, transmit, or store any data.

## Data Collection

This plugin collects **no data whatsoever**. Specifically:

- **No personal information** is collected, stored, or transmitted
- **No usage analytics or telemetry** are gathered
- **No network requests** are made by this plugin
- **No cookies, tokens, or identifiers** are created or accessed
- **No code from your projects** is sent anywhere by this plugin

## How the Plugin Works

The plugin consists entirely of static Markdown files that are read locally by Claude Code to provide contextual guidance about UI/UX design principles. All processing happens locally within your Claude Code session. The plugin does not include any executable code, MCP servers, or external service integrations.

## Third-Party Services

This plugin does not integrate with or send data to any third-party services.

## Changes to This Policy

If the plugin's functionality changes in a way that affects data handling, this privacy policy will be updated accordingly in the repository.
